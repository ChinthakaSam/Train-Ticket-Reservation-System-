Steps to deploy the app......

1. Backend of application is in "Services" folder.
2. Client(frontend) of application is in "Client" folder.
3. Frontend is called as "train-ticket-reservation-app".
4. Open cmd from "train-ticket-reservation-app" directory and run "npm start".
5. Open another 2 cmds from "backend" directory.
6. Run "mongod" in one cmd.
7. Connection is on port 27017.
8. Run "mongo" in other cmd and then run "use ticketDB" to access data base.
9. "MongoDB Data files" folder contains the some files from "\data\db" path where data stored.
